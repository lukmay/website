SmartBreeze

A prototype smart home system employs IoT devices to monitor temperature and humidity, transmitting this data to a server. The server processes the information and controls the airflow in a building by automatically opening and closing windows using a novel pressure principle. This innovative approach adapts to any window without modifications. Developed during the "Servus Zukunft Hackathon" 2024

Features :

  • Innovative and cost-effective pressure principle.
  • Open API using Spring Boot.
  • User-friendly web interface for data analysis.
